# 🏦 Credit Risk Optimization Engine (Power BI Project)

## 📌 Overview
This project presents a fintech-style Credit Risk Optimization Engine built using Power BI.  
It transforms raw credit card data into a structured analytical system that evaluates customer risk, estimates expected losses, and supports credit decision-making.

---

## 🎯 Objectives
- Assess customer-level default risk
- Estimate expected loss across the portfolio
- Identify high-risk segments
- Support decision-making (Increase, Maintain, Reduce credit exposure)

---

## 🧠 Key Features
- Data transformation using Power Query (Unpivoting, Merging, Cleaning)
- Data modeling with fact and dimension tables
- Risk feature engineering:
  - Default Probability
  - Expected Loss
  - Customer Segmentation
- Decision Engine for credit optimization
- Interactive fintech-style dashboard

## 📈 Key Insights
- Average Default Probability: ~28%
- High-risk customers contribute the majority of expected losses
- Portfolio shows low utilization but high potential risk exposure
- Risk is concentrated in specific customer segments
